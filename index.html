Web Ergonomics Principles

Applied to the Learning Content Management System

1. Readability & Contrast

Using high contrast ratios between text and background ensures content is legible for all students, including those with visual impairments.

2. Consistency & Familiarity

Standard navigation patterns and layout consistency reduce cognitive load, helping students focus on the learning material.

3. Accessibility (Perceivability)

Using semantic HTML5 elements ensures that screen readers and other assistive technologies can correctly parse the system's structure.

4. Responsiveness (Flexibility)

The fluid 3:9 grid system ensures the LCMS is usable on desktops, tablets, and smartphones without loss of information.

Deployment to Apache

To move the completed LCMS website from your home directory to the Apache document root, execute the following commands in your terminal:

# 1. Copy files to Apache document root

sudo cp -r ~/web-ui-exam/* /var/www/html/

# 2. Adjust ownership to the Apache user (www-data)

sudo chown -R www-data:www-data /var/www/html/

# 3. Set directory permissions (755)

sudo find /var/www/html/ -type d -exec chmod 755 {} \;

# 4. Set file permissions (644)

sudo find /var/www/html/ -type f -exec chmod 644 {} \;
